# Netflix Clone
A clone web application of Netflix .<br/>
Deployed App : https://netflix-b824e.web.app/

## Technologies Used
React
React-Router
Redux
Tailwind CSS

## How to run locally?
Clone the repo
```bash
https://github.com/i-nithin/netflix-clone.git
```
Go to project directory
```bash
cd netflix-clone
```
Install dependencies
```bash
npm install
```
Rut the App
```bash
npm start
```
